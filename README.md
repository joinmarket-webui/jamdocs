# Jam Docs | [jamdocs.org](https://jamdocs.org/)

Documentation for [Jam](https://jamapp.org/). Built using mkdocs material theme.

### Install
```
pip install mkdocs
pip install mkdocs-material
```

### Run
```
mkdocs serve
```

### Build
```
mkdocs build
```
